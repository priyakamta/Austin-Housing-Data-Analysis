# 🏡 Austin Housing Data Insights Dashboard

**Author:** Preety

---

## 🚀 Project Overview

This project presents a **comprehensive real estate analytics dashboard** built using Power BI to analyze housing data in Austin, Texas.

The dashboard enables users to explore **property pricing, location insights, home features, and market trends** through interactive visualizations.

It simulates a real-world business scenario where data is used to support **property investment and buying decisions**.

---

## 🛠️ Tech Stack

* Microsoft Power BI
* Data Visualization
* Real Estate Analytics
* Data Analysis

---

## 📊 Dataset

The dataset contains detailed housing information including:

* Property prices
* Living area and lot size
* Location (City, Zip Code)
* Home type and features
* Year of construction

## 📊 Key Insights

* Majority of properties belong to **Single Family category**
* Significant variation between **median and average home prices**
* Larger living area correlates with higher property prices
* Certain zip codes show **higher property concentration**
* Property features (cooling, heating, etc.) influence pricing trends
